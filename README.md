 Project 3 — SQL Data Analysis
Project Overview
This project involves querying a cleaned e-commerce sales dataset using SQL Server Management Studio (SSMS) to extract meaningful business insights. The goal was to use SQL to analyze patterns, trends, and performance metrics across 1,200 orders.

Objectives
	•	Write SELECT queries to explore the dataset
	•	Use WHERE clause to filter specific records
	•	Apply ORDER BY to sort and rank results
	•	Use GROUP BY with aggregation functions — COUNT, SUM, AVG
	•	Identify top-performing products, referral channels, and coupon effectiveness

Columns used:
OrderID, Date, CustomerID, Product, Quantity, UnitPrice, ShippingAddress, PaymentMethod, OrderStatus, TrackingNumber, ItemsInCart, CouponCode, ReferralSource, TotalPrice.
